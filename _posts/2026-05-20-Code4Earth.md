A proposal led by Judongyang (Judy) Zhou has been selected for one of the ECMWF Code for Earth Challanges: [*AI-Based Uncertainty-Aware Air Quality Assessment*](https://github.com/ECMWFCode4Earth/Challenges_2026/issues/4). The challange is brought up by ECMWF researchers that has practical significance and open for proposal. 

The project aims to look at the uncertainty in daily atmosphere components forecast by training a model to learn the error between analysis and forecast. The model would help quantify the AQI forecast and generate AQI reports automatically. Please check the webpage for the selected project proposed by Judy: [AIr-wise](https://codeforearth.ecmwf.int/project/air-wise/).

Congratulations to Judy and Linfeng!
