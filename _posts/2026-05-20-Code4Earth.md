A proposal led by Judongyang (Judy) Zhou has been selected for one of the ECMWF Code for Earth Challenges: [*AI-Based Uncertainty-Aware Air Quality Assessment*](https://github.com/ECMWFCode4Earth/Challenges_2026/issues/4). The challenge was proposed by ECMWF researchers and focuses on problems with practical significance that are open for community proposals.

The project aims to investigate uncertainty in daily atmospheric component forecasts by training a model to learn the error between analyses and forecasts. The model will help quantify uncertainty in AQI forecast and automatically generate AQI reports. More information about the selected project is available here: [AIr-wise](https://codeforearth.ecmwf.int/project/air-wise/).

Congratulations to team members Judy and Linfeng!
